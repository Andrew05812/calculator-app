# Команды для мерджа изменений в GitHub

## Если репозиторий уже инициализирован:

```bash
# 1. Проверить статус изменений
git status

# 2. Добавить все измененные и новые файлы
git add .

# 3. Или добавить конкретные файлы
git add style.css index.html setup_project/

# 4. Создать коммит с описанием изменений
git commit -m "Улучшен интерфейс калькулятора: добавлен современный дизайн с градиентами и анимациями"

# 5. Отправить изменения в GitHub
git push origin main
```

## Если репозиторий не инициализирован:

```bash
# 1. Инициализировать git репозиторий
git init

# 2. Добавить удаленный репозиторий (замените URL на ваш)
git remote add origin https://github.com/Andrew05812/calculator-app.git

# 3. Добавить все файлы
git add .

# 4. Создать первый коммит
git commit -m "Улучшен интерфейс калькулятора: добавлен современный дизайн с градиентами и анимациями"

# 5. Установить ветку main (если нужно)
git branch -M main

# 6. Отправить в GitHub
git push -u origin main
```

## Если нужно обновить существующий репозиторий:

```bash
# 1. Получить последние изменения с GitHub
git pull origin main

# 2. Разрешить конфликты (если есть)

# 3. Добавить изменения
git add .

# 4. Создать коммит
git commit -m "Улучшен интерфейс калькулятора: добавлен современный дизайн с градиентами и анимациями"

# 5. Отправить изменения
git push origin main
```

## Для PowerShell (Windows):

```powershell
# 1. Проверить статус
git status

# 2. Добавить все файлы
git add .

# 3. Создать коммит
git commit -m "Улучшен интерфейс калькулятора: добавлен современный дизайн с градиентами и анимациями"

# 4. Отправить в GitHub
git push origin main
```
