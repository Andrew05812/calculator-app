-----------------------------------------------------------
УСТАНОВКА
-----------------------------------------------------------

1. Запустите calculator-app-setup-1.0.0.exe
2. Следуйте инструкциям мастера установки
3. Выберите папку установки (по умолчанию: C:\Program Files\Calculator App)
4. Нажмите "Install"
5. Дождитесь завершения установки

-----------------------------------------------------------
УДАЛЕНИЕ
-----------------------------------------------------------

Способ 1: Через меню Пуск
  Пуск → Calculator App → Uninstall Calculator App

Способ 2: Через Панель управления
  Панель управления → Программы и компоненты → Calculator App → Удалить

-----------------------------------------------------------
ФУНКЦИИ ПРИЛОЖЕНИЯ
-----------------------------------------------------------

Базовые операции:
  - Сложение (+)
  - Вычитание (-)
  - Умножение (×)
  - Деление (÷)
  - Модуло (%)

Научные функции:
  - Sin (синус)
  - Cos (косинус)
  - Возведение в степень (x^y)
  - Квадратный корень (√)
  - Округление вниз (floor)
  - Округление вверх (ceil)

Дополнительно:
  - Память: M+ (добавить), MC (очистить), MR (восстановить)
